# 今日の積み上げ

# link_toのanchorオプションとは?

リンク先にページ遷移をした際、場所(位置)を指定することができる。

# 実装

```azure
<%= link_to 'リンクのテキスト', リンク先のページのpath(anchor: 'リンク先のページの途中のID'), class: 'aタグのクラス名' %>
```
遷移先の位置を指定するには、**ID**が必要である。
