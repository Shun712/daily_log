# deviseでログイン有無のコントローラ設定

deviseのヘルパーメソッドで`authenticate_user!`使い、ユーザーのログイン済みか判断させる。

# ログインしたユーザーのみアクションが実行できるよう設定

`before_action :authenticate_user!`

# ログインをスキップさせた場合

`skip_before_action :authenticate_user!`
